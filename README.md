# Formulário de E-mail Javascript

Projeto desenvolvido na disciplina de Desenvolvimento em Javascript da faculdade.

## Objetivo
Criar um formulário simples para validação de e-mail utilizando HTML, CSS e JavaScript.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Campo para inserção de e-mail
- Validação automática do e-mail
- Verificação do caractere @
- Mensagens de alerta para erro
- Mensagem de sucesso ao enviar

## Plataforma utilizada
- Playcode.io
